# Project: Real-Time Ping Pong Game

This project is a terminal-based ping pong game using **Pygame**. It introduces an interactive game design using object-oriented principles and real-time graphical rendering.



## Folder Structure

```
pygame-pingpong/
├── main.py
├── requirements.txt
├── game/
│   ├── game_engine.py
│   ├── paddle.py
│   └── ball.py
└── README.md
```
